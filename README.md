# Mvc5-Dynamic-CV-Portfolio-
ASP.NET MVC5 ile geliştirilmiş, admin panel üzerinden tüm içeriklerin dinamik olarak yönetilebildiği CV &amp; portföy web uygulaması. 

MVC5 Dinamik CV & Portföy Projesi

Bu proje, ASP.NET MVC 5 kullanılarak geliştirilmiş dinamik bir CV & portföy web uygulamasıdır.
Proje içerisinde tüm içerikler admin panel üzerinden yönetilebilir yapıdadır.

Kullanılan Teknolojiler
ASP.NET MVC 5
C#
Entity Framework (Database First)
SQL Server
HTML, CSS, Bootstrap

Proje Özellikleri
Hakkımda, Deneyimler, Eğitimler, Yetenekler, Sertifikalar ve İletişim bölümleri
Admin panel üzerinden tüm içeriklerin eklenmesi, güncellenmesi ve silinmesi (CRUD)
Veritabanı ile dinamik içerik yönetimi
Responsive (mobil uyumlu) tasarım
Kullanıcı tarafı ile admin panelin entegre çalışması

Projenin Amacı
Bu proje ile:
MVC mimarisini daha iyi anlamak
Veritabanı işlemlerini yönetmek
Admin panel geliştirme deneyimi kazanmak
amaçlanmıştır.
